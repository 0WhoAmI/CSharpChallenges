# 🚀 C# Daily Challenges & Algorithms
A personal repository dedicated to mastering C#, algorithms, and data structures through daily coding challenges. 

This project is not just about finding the right answers, but about writing **clean, optimized, and idiomatic C# code** backed by automated unit tests.

## 🎯 Goals
* **Consistency:** Solving at least one algorithmic problem daily.
* **Performance:** Focusing on Time and Space Complexity (Big O notation).
* **Quality:** Writing production-ready code using SOLID principles and meaningful naming conventions.
* **Testing:** TDD approach with comprehensive unit tests for every solution.

## 🛠️ Tech Stack & Architecture
* **Language:** C# 14 / .NET 10
* **Testing Framework:** xUnit
* **Structure:** Clean Solution architecture separating core algorithms (`src`) from test cases (`tests`).

## 📂 Project Structure
The repository is organized by platform and difficulty level to maintain clarity as the project scales:
```
CSharpChallenges/
 ├── src/CSharpChallenges/        # Core logic & algorithms
 │    └── LeetCode/Easy/
 └── tests/CSharpChallenges.Tests/ # Unit tests (xUnit)
      └── LeetCode/Easy/
```

## 🧪 How to run tests
To execute all test cases, navigate to the root directory of the repository and run:
```
dotnet test
```
